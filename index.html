Tithe

Tithe is a modern web application for contributing to your favorite charity. Tithe is sourced by the Charity Navigator, Unsplashed, and Google Maps Embed APIs. It allows the user to choose a state and get a list of charities back for that state. Users can save and “spotlight” charities they like in a section that persists when the users leave the site. Tithe filters the charities returned by the Charity Navigator API to only return rated charities.
